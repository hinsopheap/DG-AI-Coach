# Public assets

Save your coach portrait photo here as **`coach.jpg`** (JPEG or PNG works).

The landing page (`pages/index.js`) references `/coach.jpg` for the coach
hero card. If the file is missing, the site falls back to a styled initials
avatar automatically — no visual breakage either way.

Recommended specs:
- Square crop, 600×600 or larger
- White or neutral background
- JPEG, 70-85% quality, under 200KB

Drop the file in, redeploy, and it will render on the landing page and
optionally in the chat interface as the coach avatar.
